# 🛒 MERN E-commerce Website

A simple and functional e-commerce web application built using the MERN stack. This project allows users to browse products, view details, and add items to a shopping cart.

---

## 🚀 Features

* 🛍️ Product listing with images, price, and description
* ➕ Add to Cart functionality
* 🛒 Cart display with selected items
* 🎨 Clean and responsive UI
* 📦 Data stored using MongoDB

---

## 🛠️ Tech Stack

* **Frontend:** React.js
* **Backend:** Node.js, Express.js
* **Database:** MongoDB
* **HTTP Client:** Axios

## ⚙️ How to Run Locally

### 1️⃣ Clone the repository

```
git clone https://github.com/your-username/your-repo-name.git
cd ecommerce-app
```

---

### 2️⃣ Setup Backend

```
cd backend
npm install
node server.js
```

---

### 3️⃣ Setup Frontend

```
cd frontend
npm install
npm start
```

## 🌐 API Endpoints

* **GET /products** → Fetch all products
* **POST /products** → Add new product
